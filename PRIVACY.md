# Privacy Policy — AIClip

**Last updated:** June 29, 2026

## Overview

AIClip is a Chrome extension that clips and saves AI chatbot responses. This privacy policy explains how AIClip handles user data.

## Data Collection

AIClip does **not** collect, transmit, or share any user data.

- No personal information is collected
- No browsing history is tracked
- No analytics or telemetry data is sent
- No cookies are set
- No external API calls are made
- No remote servers are contacted

## Data Storage

All data (saved clips, notebooks, and preferences) is stored exclusively in Chrome's built-in sync storage API. This data lives within the user's Chrome browser profile and syncs across devices only if the user is signed into Chrome.

## Data Usage

Saved clips are used solely to display them in the extension, insert them into AI chatbot input fields, and sync across the user's own devices.

## Third-Party Services

AIClip does not integrate with any third-party services, APIs, or servers.
