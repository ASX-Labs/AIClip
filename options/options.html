About AIClip

Clip and save AI chatbot responses with one click.

VERSION

AIClip v1.0.0

HOW IT WORKS

When you visit a supported AI chatbot, a Clip button appears on each AI response. Click it to save the response to your library. You can also select any text and right-click to clip it.

KEYBOARD SHORTCUT

Press Alt + C on any supported AI site to open the clips overlay.

EXPORT FORMATS

JSON — full backup with metadata, importable back into AIClip.
Markdown — human-readable format for notes, docs, or sharing.

SYNC

Clips sync automatically across all devices where you're signed into Chrome.

PRIVACY

All data is stored locally in Chrome sync storage. No external APIs, no servers, no tracking.
